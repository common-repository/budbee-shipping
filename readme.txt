== Description ==

First class shipping in Stockholm, Gothenburg and Malmoe. Customers can track their parcels and communicate directly with the driver.

= BudBee staff will always ask the customer for =

**We will always ask for door code.** The customer will always be able to update information to the driver. Even if the door code changes the last minute before delivery.

**We always deliver on the customers terms.** Through an intuitive web app the customer will be able to update important information to the driver.

**Unsuccessful delivery.**  Anything could happen in life. Relax! There is always an option to reschedule delivery.

= Minimum requirements =
* You need an account with Budbee.
Budbee only works in Stockholm, Gothenburg and Malmoe.

== Installation ==

From your WordPress dashboard

1. **Visit** Plugins > Add New
2. **Search** for "Budbee"
3. **Activate** the plugin on your plugin page in WordPress 
4. **Add a shipping method** and name it Budbee or what ever suits you :-)
5. **Add API keys** Add the API-key and API-secret that you received from Budbee [get started](https://www.budbee.com/)
